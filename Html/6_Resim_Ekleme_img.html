Resim Etiketleri : <img>

<img> : Sayfaya görsel (resim) eklemek için kullanılır. Boş (self-closing) bir etikettir, yani kapanış etiketi yoktur.
<img> Özellikleri ve Değerleri
Özellik Tanım Değer Tanım
src Görüntünün URL adresini belirtir. URL Görüntünün URL adresini belirtir.
alt Görüntü yerine alternatif metin belirtir. text Görüntü yerine alternatif metin belirtir.
height Görüntünün pikselcinsinden yüksekliğini belirtir. pixels Görüntünün pikselcinsinden yüksekliğini belirtir.
width Görüntünün piksel cinsinden genişliğini belirtir. pixels Görüntünün pikselcinsinden genişliğini belirtir.
usemap İstemci taraflı resim haritası olduğunu belirtir. #mapname <map> etiketinin name veya id özelliğini belirtir.
