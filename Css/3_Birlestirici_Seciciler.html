Bi̇rleşti̇ri̇ci̇ Seçi̇ci̇ler (Combi̇nators)

Alt Seçicler (Boşluk) : Bir HTML öğesi altındaki seçilmek istenen öğeleri belritmek için kullanılır.
Alt Seçici
HTML
CSS
Çocuk (>) Seçiciler : Bir HTML öğesinin hemen altındaki ilk hiyerarşiden gelen öğeleri seçmek için kullanılır. Seçilen öğelerin altındaki aynı tür elemanları farklı şekilde stillendirmek istiyorsak çocuk seçici > kullanmak büyük fayda sağlayacaktır.
Çocuk Seçici
HTML
CSS
Komşu Kardeş Seçici (+) : Komşu kardeş seçiciler genellikle belirli bir öğeden sonra gelen aynı hiyerarşideki ilk öğeyi seçmek için kullanılır. Aşağıdaki örnekte h4 öğesinden sonra gelen ilk p öğesi seçilmektedir.
Komşu Kardeş Seçici
HTML
CSS
Genel Seçici (~) : Genel seçici ~ ile bir öğeden sonra gelen tüm kardeş öğeler seçilebilir.
Genel Seçici
HTML
CSS
