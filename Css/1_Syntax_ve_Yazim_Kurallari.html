Css Şartları ve Kuralları

  • CSS kodları sayfa içerisinde kullanılıyor ise mutlaka style tag'ı (etiketi) içerisinde yazılmalıdır.
  • CSS kodları sayfa içerisinde style tag'ı (etiketi) içerisinde kullanılıyorsa her seçici {} (süslü parantezler) içerisinde yazılmalıdır.
  • CSS kodları sayfa içerisinde ve satır içi kullanılıyor ise mutlaka style global özelliği içerisinde yazılmalıdır.
  • CSS kodları sayfa içerisinde ve satır içi class (sınıf) şeklinde kullanılıyor ise mutlaka class global özelliği içerisine dahil edilmelidir.
  • CSS'de özellik ve değer arasında mutlaka : (iki nokta) karakteri eklenmelidir.
  • CSS'de özellik ve değer kullanıldıktan sonra mutlaka ; (noktalı virgül) ile sonlandırılmalıdır.
  • CSS'de birden fazla tag'a (etikete) erişilmek isteniyorsa mutlaka taglar arasına , (virgül) eklenmelidir.
  • Sayfaya harici bir CSS dosyasını dahil edilmek isteniyor ise link HTML tag'ını kullanılmalıdır ve ilgili CSS dosyası içerisinde style tag'ları (etiketleri) kullanılmamalıdır.
