Sass String

Function Description & Example
quote(string) Dizeye tırnak işaretleri ekler ve sonucu döndürür.

Example:
quote(Hello world!)
Result: "Hello world!"
str-index(string, substring) Alt dizenin dize içindeki ilk oluşumunun dizinini döndürür.

Example:
str-index("Hello world!", "H")
Result: 1
str-insert(string, insert, index) Belirtilen dizin konumuna ekleme eklenmiş dizeyi döndürür.

Example:
str-insert("Hello world!", " wonderful", 6)
Result: "Hello wonderful world!"
str-length(string) Dizenin uzunluğunu (karakter cinsinden) döndürür.
Example:
str-length("Hello world!")
Result: 12
str-slice(string, start, end) Dizeden karakterleri çıkarır; başlangıçta başlayıp bitişte bitir ve dilimi döndürür.

Example:
str-slice("Hello world!", 2, 5)
Result: "ello"
to-lower-case(string) Küçük harfe dönüştürülmüş dizenin bir kopyasını döndürür.

Example:
to-lower-case("Hello World!")
Result: "hello world!"
to-upper-case(string) Büyük harfe dönüştürülmüş dizenin bir kopyasını döndürür.

Example:
to-upper-case("Hello World!")
Result: "HELLO WORLD!"
unique-id() Rastgele oluşturulmuş, tırnak içine alınmamış benzersiz bir dize döndürür (mevcut küstahlık oturumunda benzersiz olduğu garanti edilir).

Example:
unique-id()
Result: tyghefnsv
unquote(string) Dize etrafındaki tırnak işaretlerini (varsa) kaldırır ve sonucu döndürür.
Example:
unquote("Hello world!")
Result: Hello world!

Sass List

Function Description & Example
append(list, value, [separator]) Listenin sonuna tek bir değer ekler. ayırıcı otomatik, virgül veya boşluk olabilir. otomatik varsayılandır.

Example:
append((a b c), d)
Result: a b c d
append((a b c), (d), comma)
Result: a, b, c, d
index(list, value) Listedeki değerin dizin konumunu döndürür.

Example:
index(a b c, b)
Result: 2
index(a b c, f)
Result: null
is-bracketed(list) Listenin köşeli parantez içerip içermediğini kontrol eder.

Example:
is-bracketed([a b c])
Result: true
is-bracketed(a b c)
Result: false
join(list1, list2, [separator, bracketed]) Liste2'yi liste1'in sonuna ekler. ayırıcı otomatik, virgül veya boşluk olabilir. auto varsayılandır (ilk listedeki ayırıcıyı kullanacaktır). Parantez içine alma işlemi otomatik, doğru veya yanlış olabilir. otomatik varsayılandır.

Example:
join(a b c, d e f)
Result: a b c d e f
join((a b c), (d e f), comma)
Result: a, b, c, d, e, f
join(a b c, d e f, $bracketed: true)
Result: [a b c d e f]
length(list) Listenin uzunluğunu döndürür.

Example:
length(a b c)
Result: 3
list-separator(list) Kullanılan liste ayırıcıyı dize olarak döndürür. Boşluk veya virgül olabilir.

Example:
list-separator(a b c)
Result: "space"
list-separator(a, b, c)
Result: "comma"
nth(list, n) Listedeki n'inci öğeyi döndürür.

Example:
nth(a b c, 3)
Result: c
set-nth(list, n, value) N'inci liste öğesini belirtilen değere ayarlar.

Example:
set-nth(a b c, 2, x)
Result: a x c
zip(lists) Listeleri tek bir çok boyutlu liste halinde birleştirir.

Example:
zip(1px 2px 3px, solid dashed dotted, red green blue)
Result: 1px solid red, 2px dashed green, 3px dotted blue

Sass Map

Function Description & Example
map-get(map, key) Haritada belirtilen anahtarın değerini döndürür.

Example:
$font-sizes: ("small": 12px, "normal": 18px, "large": 24px)
map-get($font-sizes, "small")
Result: 12px
map-has-key(map, key) Haritanın belirtilen anahtara sahip olup olmadığını kontrol eder. Doğru veya yanlış döndürür.

Example:
$font-sizes: ("small": 12px, "normal": 18px, "large": 24px)
map-has-key($font-sizes, "big")
Result: false
map-keys(map) Haritadaki tüm anahtarların listesini döndürür.

Example:
$font-sizes: ("small": 12px, "normal": 18px, "large": 24px)
map-keys($font-sizes)
Result: "small", "normal, "large"
map-merge(map1, map2) Harita2'yi harita1'in sonuna ekler.

Example:
$font-sizes: ("small": 12px, "normal": 18px, "large": 24px)
$font-sizes2: ("x-large": 30px, "xx-large": 36px)
map-merge($font-sizes, $font-sizes2)
Result: "small": 12px, "normal": 18px, "large": 24px, "x-large": 30px, "xx-large": 36px
map-remove(map, keys...) Belirtilen anahtarları haritadan kaldırır.

Example:
$font-sizes: ("small": 12px, "normal": 18px, "large": 24px)
map-remove($font-sizes, "small")
Result: ("normal": 18px, "large": 24px)
map-remove($font-sizes, "small", "large")
Result: ("normal": 18px)
map-values(map) Haritadaki tüm değerlerin bir listesini döndürür.
Example:
$font-sizes: ("small": 12px, "normal": 18px, "large": 24px)
map-values($font-sizes)
Result: 12px, 18px, 24px

Sass Selector

Function Description & Example
is-superselector(super, sub) Süper seçicinin alt eşleşen tüm öğelerle eşleşip eşleşmediğini kontrol eder.
Example:
is-superselector("div", "div.myInput")
Result: true
is-superselector("div.myInput", "div")
Result: false
is-superselector("div", "div")
Result: true
selector-append(selectors) İkinci (ve üçüncü/dördüncü vb.) seçiciyi birinci seçiciye ekler.

Example:
selector-append("div", ".myInput")
Result: div.myInput
selector-append(".warning", "__a")
Result: .warning__a
selector-extend(selector, extendee, extender)  
selector-nest(selectors) Sağlanan listeye göre CSS seçicilerinin iç içe geçmiş listesini içeren yeni bir seçici döndürür.

Example:
selector-nest("ul", "li")
Result: ul li
selector-nest(".warning", "alert", "div")
Result: .warning div, alert div
selector-parse(selector) Üst seçiciyle aynı formatı kullanarak seçicide bulunan dizelerin listesini döndürür.

Example:
selector-parse("h1 .myInput .warning")
Result: ('h1' '.myInput' '.warning')
selector-replace(selector, original, replacement) Orijinalde belirtilen seçicilerin yerine, belirtilen seçicilerin yer aldığı yeni bir seçici döndürür.

Example:
selector-replace("p.warning", "p", "div")
Result: div.warning
selector-unify(selector1, selector2) Yalnızca hem seçici1 hem de seçici2 tarafından eşleşen öğelerle eşleşen yeni bir seçici döndürür.

Example:
selector-unify("myInput", ".disabled")
Result: myInput.disabled
selector-unify("p", "h1")
Result: null
simple-selectors(selectors) Seçicilerdeki tek tek seçicilerin listesini döndürür.

Example:
simple-selectors("div.myInput")
Result: div, .myInput
simple-selectors("div.myInput:before")
Result: div, .myInput, :before

Sass Set Color

Function Description & Example
rgb(red, green, blue) Kırmızı-Yeşil-Mavi (RGB) modelini kullanarak bir renk ayarlar. Bir RGB renk değeri şu şekilde belirtilir: rgb(kırmızı, yeşil, mavi). Her parametre o rengin yoğunluğunu tanımlar ve 0 ile 255 arasında bir tam sayı veya bir yüzde değeri (%0 ile %100 arasında) olabilir.

Example:
rgb(0, 0, 255); // rendered as blue because the blue parameter is set to its highest value (255) and the others are set to 0
rgba(red, green, blue, alpha) Kırmızı-Yeşil-Mavi-Alfa (RGBA) modelini kullanarak bir renk ayarlar. RGBA renk değerleri, rengin opaklığını belirten alfa kanalına sahip RGB renk değerlerinin bir uzantısıdır. Alfa parametresi 0,0 (tamamen şeffaf) ile 1,0 (tamamen opak) arasında bir sayıdır.

Example:
rgba(0, 0, 255, 0.3); // rendered as blue with opacity
hsl(hue, saturation, lightness) Ton-Doygunluk-Açıklık (HSL) modelini kullanarak bir renk ayarlar ve renklerin silindirik koordinat gösterimini temsil eder. Ton, renk tekerleğinde bir derecedir (0'dan 360'a kadar) - 0 veya 360 kırmızı, 120 yeşil, 240 mavidir. Doygunluk bir yüzde değeridir; %0 grinin bir tonu, %100 ise tam renk anlamına gelir. Hafiflik de bir yüzdedir; %0 siyah, %100 beyazdır.

Example:
hsl(120, 100%, 50%); // green
hsl(120, 100%, 75%); // light green
hsl(120, 100%, 25%); // dark green
hsl(120, 60%, 70%); // pastel green
hsla(hue, saturation, lightness, alpha) Ton-Doygunluk-Açıklık-Alfa (HSLA) modelini kullanarak bir renk ayarlar. HSLA renk değerleri, rengin opaklığını belirten alfa kanalına sahip HSL renk değerlerinin bir uzantısıdır. Alfa parametresi 0,0 (tamamen şeffaf) ile 1,0 (tamamen opak) arasında bir sayıdır.

Example:
hsl(120, 100%, 50%, 0.3); // green with opacity
hsl(120, 100%, 75%, 0.3); // light green with opacity
grayscale(color) Renkle aynı açıklığa sahip gri bir renk ayarlar.

Example:
grayscale(#7fffd4);
Result: #c6c6c6
complement(color) Rengin tamamlayıcı rengi olan bir renk ayarlar.

Example:
complement(#7fffd4);
Result: #ff7faa
invert(color, weight) Rengin tersi veya negatif rengi olan bir renk ayarlar. Ağırlık parametresi isteğe bağlıdır ve %0 ile %100 arasında bir sayı olmalıdır. Varsayılan %100'dür.

Example:
invert(white);
Result: black

Sass Manipulate Color

Function Description & Example
mix(color1, color2, weight) Renk1 ve Renk2 karışımından oluşan bir renk oluşturur. Ağırlık parametresi %0 ile %100 arasında olmalıdır. Daha büyük ağırlık, daha fazla rengin1 kullanılması gerektiği anlamına gelir. Daha küçük ağırlık, daha fazla rengin2 kullanılması gerektiği anlamına gelir. Varsayılan %50'dir.
adjust-hue(color, degrees) Rengin tonunu -360 dereceden 360 dereceye kadar bir dereceyle ayarlar.

Example:
adjust-hue(#7fffd4, 80deg);
Result: #8080ff
adjust-color(color, red, green, blue, hue, saturation, lightness, alpha) Bir veya daha fazla parametreyi belirtilen miktara göre ayarlar. Bu işlev, belirtilen miktarı mevcut renk değerine ekler veya çıkarır.

Example:
adjust-color(#7fffd4, blue: 25);
Result:
change-color(color, red, green, blue, hue, saturation, lightness, alpha) Bir rengin bir veya daha fazla parametresini yeni değerlere ayarlar.

Example:
change-color(#7fffd4, red: 255);
Result: #ffffd4
scale-color(color, red, green, blue,  saturation, lightness, alpha) Bir veya daha fazla renk parametresini ölçeklendirir.
rgba(color, alpha) Verilen alfa kanalıyla yeni bir renk rengi oluşturur.
Example:
rgba(#7fffd4, 30%);
Result: rgba(127, 255, 212, 0.3)
lighten(color, amount) %0 ila %100 arasında bir miktarda daha açık bir renk rengi oluşturur. Miktar parametresi HSL açıklığını bu yüzde oranında artırır.
darken(color, amount) %0 ile %100 arasında bir miktarda daha koyu bir renk rengi oluşturur. Miktar parametresi HSL açıklığını bu yüzde oranında azaltır.
saturate(color, amount) %0 ile %100 arasında bir miktarda daha doygun bir renk rengi oluşturur. Miktar parametresi HSL doygunluğunu bu yüzde oranında artırır.
desaturate(color, amount) %0 ile %100 arasında bir miktarda daha az doygun bir renk rengi oluşturur. Miktar parametresi HSL doygunluğunu bu yüzde oranında azaltır.
opacify(color, amount) 0 ile 1 arasındaki miktarda daha opak bir renk rengi oluşturur. Miktar parametresi alfa kanalını o miktarda artırır.
fade-in(color, amount) 0 ile 1 arasındaki miktarda daha opak bir renk rengi oluşturur. Miktar parametresi alfa kanalını bu miktarda artırır.
transparentize(color, amount) 0 ile 1 arasındaki miktarda daha şeffaf bir renk rengi oluşturur. Miktar parametresi alfa kanalını o kadar azaltır.
fade-out(color, amount) 0 ile 1 arasındaki miktarda daha şeffaf bir renk rengi oluşturur. Miktar parametresi alfa kanalını o kadar azaltır.
