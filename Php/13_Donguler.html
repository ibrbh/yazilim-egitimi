Ana Sayfa

Döngüler

While

Bir koşul true olduğu sürece kod bloğunu tekrar tekrar çalıştıran döngüdür.
<?php $sayi = 1; while ($sayi <= 5) { echo $sayi; $sayi++; } ?>

Do While

Kod bloğunu önce bir kez çalıştırıp ardından koşulu kontrol eden döngü yapısıdır.
<?php $sayi = 1; do { echo $sayi; $sayi++; } while ($sayi <= 5); ?>

For

Belirli bir başlangıç, bitiş ve artış değerine göre çalışan döngüdür.
for (başlangıç; koşul; arttırma) { ... }
<?php for ($i = 1; $i <= 5; $i++) { echo $i; } ?>

Foreach

Bir dizinin elemanlarını anahtar–değer şeklinde veya sadece değer olarak tek tek dönen döngüdür.
<?php $meyveler = ["Elma", "Armut", "Kiraz"]; foreach ($meyveler as $meyve) { echo $meyve; } ?>
<?php $kisiler = ["ad" => "Ali", "yas" => 25]; foreach ($kisiler as $key => $value) { echo "$key : $value"; } ?>

Continue

O anki döngü adımını atlayıp döngünün bir sonraki adımına geçer.
<?php for ($i = 1; $i <= 10; $i++) { if ($i == 5) { continue; // 5 yazılmadan döngü devam eder } echo $i; } ?>

Break

Döngüyü tamamen durduran komuttur.
<?php for ($i = 1; $i <= 10; $i++) { if ($i == 5) { break; // 5 yazılmadan döngü devam eder } echo $i; } ?>
