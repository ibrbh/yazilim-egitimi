Strings Metodları

strlen() : Metnin karakter sayısını döndürür.
str_word_count() : Metindeki kelime sayısını döndürür.
strtolower() : Metnin bütün harflerini küçük harfe çevirir.
strtoupper() : Metnin bütün harflerini büyük harfe çevirir.
ucfirst() : String ifadenin ilk harfini büyük harfe çevirir.
str_replace() : String ifadedeki bazı karakterleri değiştirir.
strrev() : String ifadeyi tersine çevirir.
strpos() : Bir string ifadenin başka bir string ifade içinde kaçıncı index değerinde olduğunu döndürür.
<?php echo strlen("Hello world!"); // outputs 12 echo str_word_count("Hello world!"); // outputs 2 echo strtolower("Hello WORLD."); // outputs hello world echo strtoupper("Hello WORLD!"); // outputs HELLO WORLD echo ucfirst("hello world!"); // outputs Hello world echo str_replace("world", "Dolly", "Hello world!"); // outputs Hello Dolly! echo strrev("Hello world!"); // outputs !dlrow olleH echo strpos("Hello world!", "world"); // outputs 6 ?>
