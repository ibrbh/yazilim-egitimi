Maths Metodları

abs() : Bir sayının mutlak (pozitif) değerini verir.
ceil() : Bir sayıyı en yakın üst tam sayıya yuvarlar.
floor() : Bir sayıyı en yakın alt tam sayıya yuvarlar.
round() : Kayan noktalı bir sayıyı yuvarlar.
sqrt() : Bir sayının karekökünü verir.
max() : Bir dizideki en yüksek değeri veya belirtilen birkaç değerin en yüksek değerini döndürür.
min() : Bir dizideki en düşük değeri veya belirtilen birkaç değerin en düşük değerini döndürür.
pi() : π değerini döndürür.
rand() : Rastgele bir tamsayı üretir.
<?php echo(abs(-6.7)); // returns 6.7 echo(sqrt(64)); // returns 8 echo(ceil(0.60)); // returns 1 echo(ceil(0.49)); // returns 1 echo(floor(0.60)); // returns 0 echo(floor(0.49)); // returns 0 echo(round(0.60)); // returns 1 echo(round(0.49)); // returns 0 echo(min(0, 150, 30, 20, -8, -200)); // returns -200 echo(max(0, 150, 30, 20, -8, -200)); // returns 150 echo(pi()); // returns 3.1415926535898 echo(rand()); echo(rand(10, 100)); ?>
