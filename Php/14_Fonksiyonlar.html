Fonksiyonlar

Fonksiyon Tanımlama

PHP’de fonksiyon tanımlamak için function anahtar kelimesi kullanılır.

Syntax

function fonksiyonAdi() {// çalışacak kodlar}
<?php function selamVer() { echo "Merhaba!"; } selamVer(); // Fonksiyonu çağırma ?>

Parametre Alan Fonksiyon

Fonksiyon çalışırken dışarıdan değer alabilir.
<?php function topla($a, $b) { echo $a + $b; } topla(5, 3); // 8 ?>

Varsayılan (Default) Parametre

Parametre gönderilmezse kullanılacak varsayılan değer tanımlanabilir.
<?php function selam($isim = "Ziyaretçi") { echo "Hoş geldin, $isim!"; } selam(); // Hoş geldin, Ziyaretçi! selam("İbrahim"); // Hoş geldin, İbrahim! ?>

Return Kullanımı

Fonksiyon içinde hesaplanan bir değeri dışarıya döndürmek için return kullanılır.
<?php function carp($a, $b) { return $a * $b; } $sonuc = carp(4, 5); echo $sonuc; // 20 ?>

Birden Fazla Değer Döndürme

PHP fonksiyonları tek bir değer döndürür ama diziyle birden fazla değer döndürebiliriz.
<?php function bilgiler() { return ["İbrahim", 25, "Programcı"]; } list($isim, $yas, $meslek) = bilgiler(); ?>

Değişken Sayıda Parametre (Spread/...)

...$parametre ile fonksiyona sınırsız sayıda değer gönderilebilir.
<?php function toplam(...$sayılar) { return array_sum($sayılar); } echo toplam(1, 2, 3, 4); // 10 ?>

Fonksiyon İçinde Global Değişken Kullanımı

Normalde fonksiyon içinde dışarıdaki değişkenlere erişilemez. Erişmek için global kullanılır.
<?php $x = 10; function yazdir() { global $x; echo $x; } yazdir(); // 10 ?>

Anonim (İsimsiz) Fonksiyon

Fonksiyon adı yoktur; genelde değişkene atanır.
<?php $mesaj = function() { echo "Bu bir anonim fonksiyon."; }; $mesaj(); ?>

Arrow Functions (Kısa Fonksiyon)

PHP 7.4+ ile gelen, tek satırlık fonksiyon şeklidir.
<?php $carp = fn($a, $b) => $a * $b; echo $carp(3, 4); // 12 ?>
