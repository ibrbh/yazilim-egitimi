React Temel Kavramları

Komponentler (Components) : React, UI'yi komponentler adı verilen bağımsız ve tekrar kullanılabilir parçalar halinde yapılandırır. Komponentler iki şekilde tanımlanabilir: Fonksiyonel Komponentler ve Sınıf Komponentleri.
JSX (JavaScript XML) : JSX, JavaScript içerisinde HTML benzeri bir sözdizimi sağlar. JSX, React'in render metodunda kullanılır ve JavaScript kodu ile HTML'in birleşmesini sağlar.
Props : Komponentlere veri geçmek için kullanılır. Veri dışarıdan komponentlere iletilir.
State : Komponentin kendi iç durumunu saklar. Genellikle kullanıcı etkileşimlerine yanıt vermek için kullanılır.
