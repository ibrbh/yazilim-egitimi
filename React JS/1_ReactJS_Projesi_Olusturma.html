React JS Projesi Oluşturma

Vite ile React Projesi

İlk olarak terminal ekranına " npm create vite@latest " yaz. Daha sonra çıkan ekranda projenin ismini yaz ve React' ı seçip devam et. Daha sonra " npm install " yazarak node_modules dosyalarını oluştur. " npm run dev " yazarak projeyi başlat.

create-react-app ile React Projesi Oluşturma

İlk olarak terminal ekranına " npx create-react-app old_react " yaz. Burada " old_react " kısmı projenin ismidir. Daha sonra " npm start " yazarak projeyi başlat.
