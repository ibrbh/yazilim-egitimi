React JSX

JSX : JSX, JavaScript kodu içinde HTML benzeri bir sözdizimi kullanmanıza olanak tanır. Bu, bileşenleri daha okunabilir ve yazımı daha kolay hale getirir. JSX, tarayıcılar tarafından doğrudan desteklenmez; bu nedenle, JSX kodu React tarafından React.createElement çağrılarına dönüştürülür.

.js ile .jsx Farkı

Temel Özellikler

HTML Benzeri Sözdizimi : JSX, HTML benzeri bir sözdizimine sahiptir ancak bazı farklar içerir. Örneğin, class yerine className, for yerine htmlFor kullanılır.
Editör 1
JavaScript İfadeleri : JSX içinde JavaScript ifadeleri süslü parantezler {} kullanılarak eklenir. Bu, dinamik verileri JSX içinde kullanmanızı sağlar.
Editör 1
Koşullu Render Etme (Ternary Operatörü) : JSX içinde koşullu render işlemleri yapmak için JavaScript koşul ifadeleri kullanılabilir.
Editör 1
Kısa Devre Mantığı (Short-Circuiting) : && operatörünün solundaki ifade true ise sağındaki ifadeyi çalıştırır, false ise çalıştırmaz.
Editör 1
Liste Render Etme : JSX içinde listeleri render etmek için JavaScript dizileri ve map() metodunu kullanabilirsiniz.
